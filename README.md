# コエモジ (Koemoji) - 音声・動画文字起こしアプリケーション

コエモジは、OpenAI Whisperモデルを使用して音声ファイルや動画ファイルから自動的に文字起こしを行うデスクトップアプリケーションです。直感的なインターフェイスと高精度な文字起こし機能を持ち、日本語を含む多言語に対応しています。

## 特徴

- **簡単な操作**: ドラッグ＆ドロップでファイルを追加し、ボタン一つで文字起こしを開始
- **複数ファイル対応**: 動画ファイル（MP4, AVI, MOV, MKV, WMV, FLV, WebM）や音声ファイル（MP3, WAV, OGG, FLAC）に対応
- **高精度文字起こし**: OpenAI Whisperモデルによる高品質な文字起こし
- **モデル選択**: tiny, base, small, medium, largeの5種類のモデルから精度と速度のバランスを選択可能
- **多言語対応**: 日本語、英語をはじめとする多くの言語に対応
- **タイムスタンプ付き出力**: 詳細なタイムスタンプ付きの文字起こし結果を提供
- **ローカル処理**: すべての処理がローカルで行われ、インターネット接続は初回のモデルダウンロード時のみ必要

## インストール方法

1. リリースページからZIPファイルをダウンロード
2. 任意のフォルダに展開
3. `install.bat` を実行して必要なライブラリをインストール
4. デスクトップに作成されたショートカットから起動

※ Python 3.8以上がインストールされていない場合は、インストールを促すメッセージが表示されます。

## 初回起動時の注意

初回起動時には、Whisperモデルが自動的にダウンロードされます。これにはモデルサイズに応じて時間がかかる場合があります：
- tiny: 約75MB
- base: 約142MB
- small: 約466MB
- medium: 約1.5GB
- large: 約3GB

デフォルトでは「medium」モデルが使用されます。設定画面でモデルサイズを変更できます。

## 使い方

1. 「ファイル追加」ボタンをクリックして、文字起こしするファイルを選択
2. 必要に応じて「設定」ボタンから設定を変更
3. 「文字起こし開始」ボタンをクリックして処理を開始
4. 処理が完了すると、結果がデスクトップの「コエモジ_文字起こし結果」フォルダに保存されます
5. 処理結果のプレビューも表示されます

## 設定項目

- **モデルサイズ**: 文字起こし精度と処理速度のバランスを選択
  - tiny: 最も高速だが精度は低め
  - base: 高速で基本的な精度
  - small: バランスの取れた速度と精度
  - medium: 高精度だが処理時間がやや長い
  - large: 最高精度だが処理時間が長く、メモリ消費も多い
- **言語**: 文字起こし対象の言語を指定（自動検出も可能）
- **出力先**: 結果ファイルの保存先ディレクトリ

## 動作環境

- Windows 10/11
- Python 3.8以上
- 推奨: 8GB以上のRAM（largeモデル使用時は16GB以上）
- GPU対応: CUDA対応のNVIDIA GPUがあれば処理が高速化（オプション）

## 開発者向け情報

### プロジェクト構造

```
koemoji/
├── main.py            # メインアプリケーション
├── transcriber.py     # 文字起こし機能
├── ui/                # UIコンポーネント
├── utils/             # ユーティリティ関数
├── resources/         # アイコンなどのリソース
├── ffmpeg_bin/        # FFmpegバイナリ
├── install.bat        # インストーラー
├── start_koemoji.bat  # 起動スクリプト
└── requirements.txt   # 依存ライブラリリスト
```

### 依存ライブラリ

主な依存ライブラリは以下の通りです：
- openai-whisper: 音声認識エンジン
- torch: Whisperのバックエンド
- ffmpeg-python: 動画・音声ファイル処理
- pillow: 画像処理
- tkinter: GUI（Pythonに標準搭載）

### ビルド・配布方法

1. 必要なファイルが揃っていることを確認
2. 不要なキャッシュファイル（`__pycache__`など）を削除
3. 全ファイルをZIPで圧縮
4. ユーザーは展開後に`install.bat`を実行するだけで利用可能
