# コエモジ - プロジェクトブリーフ

## プロジェクト概要
コエモジは、OpenAI Whisperモデルを使用して音声ファイルや動画ファイルから自動的に文字起こしを行うデスクトップアプリケーションです。ユーザーフレンドリーなインターフェイスと高精度な文字起こし機能を持ち、日本語を含む多言語に対応しています。

## 中核機能
1. **複数ファイル形式対応**:
   - 動画ファイル（MP4, AVI, MOV, MKV, WMV, FLV, WebM）からの文字起こし
   - 音声ファイル（MP3, WAV, OGG, FLAC）からの文字起こし

2. **高精度文字起こし**:
   - OpenAI Whisperモデルによる文字起こし
   - 複数のモデルサイズから選択可能（tiny, base, small, medium, large）
   - 日本語を含む多言語対応

3. **使いやすいUI**:
   - モダンなマテリアルデザインテイストのインターフェイス
   - ドラッグ＆ドロップでのファイル追加
   - バッチ処理による複数ファイルの連続処理

4. **出力オプション**:
   - タイムスタンプ付きテキスト出力
   - 出力先ディレクトリのカスタマイズ
   - 結果のプレビューと編集

5. **パフォーマンス**:
   - GPU高速化（対応環境の場合）
   - 処理進捗の可視化
   - キャンセル機能

## 対象ユーザー
- 講義や会議の録音を文字起こしする学生や会社員
- インタビュー録音を書き起こすジャーナリストや研究者
- 動画コンテンツにキャプションを追加したいクリエイター
- 音声データの整理や分析を行う研究者

## 技術要件
- Python 3.8以上
- OpenAI Whisperモデル
- FFmpeg（音声・動画処理用）
- Tkinter（GUI）
- PyInstaller（実行可能ファイル生成用）

## 開発目標
- 直感的で使いやすいインターフェイスの提供
- 高精度な文字起こし結果の実現
- 安定した動作と適切なエラーハンドリング
- クロスプラットフォーム対応（Windows, macOS, Linux）
- 将来的な機能拡張を見据えたモジュラー設計
